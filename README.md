# Generador de Graduados (Movil)

Aplicacion web para registrar graduados y generar imagenes verticales listas para redes sociales (WhatsApp/Facebook).

## Funcionalidades

- Registro de estudiantes (nombre, modalidad, fecha, foto/avatar)
- Base de datos y storage en Supabase (gratis con persistencia)
- Lista de registros con opcion de seleccionar o eliminar
- Generacion visual de plantilla vertical tipo movil (1080x1920)
- Descarga de imagen final en PNG

## Requisitos

- Node.js 18+ recomendado
- Cuenta en Supabase

## Configuracion Supabase

1. Crear proyecto en Supabase.
2. Ejecutar este SQL en SQL Editor:

```sql
create table if not exists students (
  id bigint generated by default as identity primary key,
  full_name text not null,
  graduation_mode text not null,
  graduation_date date not null,
  avatar_path text,
  avatar_key text,
  created_at timestamptz default now()
);
```

3. Crear un bucket en Storage llamado `avatars` y hacerlo publico.
4. (Opcional) En Storage, configurar CORS para permitir lectura desde el dominio donde este la app.

## Variables de entorno

En local o en Render, define:

- `SUPABASE_URL`
- `SUPABASE_SERVICE_ROLE_KEY` (solo en el servidor)
- `SUPABASE_BUCKET=avatars`

## Ejecucion local

```bash
npm install
npm start
```

Abrir en navegador:

- http://localhost:3000

## Despliegue gratuito en Render (sin disco)

1. Conecta tu repo GitHub en Render y crea un Web Service.
2. Build: `npm install`
3. Start: `npm start`
4. Variables de entorno: `SUPABASE_URL`, `SUPABASE_SERVICE_ROLE_KEY`, `SUPABASE_BUCKET`

## Notas

- Los avatars se guardan en Supabase Storage, no en el servidor.
- La base de datos es persistente gracias a Supabase.
